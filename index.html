My Personal Projects

My ClosestPoints project is a java implementation of a divide and conquer algorithm. The program implements the recursive algorithm on locations of K-12 schools in Vermont, obtained from the VT K-12 government website. This algorithm exhibits the divide and conquer paradigm, uitilizing arrays and array lists

  • Check Out Closest Points

Battleship Game

This project is a java implementation of a one-player Battleship game. The player enters a coordinate (on a 10x10 board) that will be determined a hit or miss. The player wins when all coordinates containing ships have been hit, meaning all ships have sunk.

  • Check out Battleship

Cargo Train

This java project uses linked lists to load and unload a theoretical cargo train, sorting objects into cargo types and destinations along the way.
